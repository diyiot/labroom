### Τι είναι

Το **Labroom** είναι μια πλατφόρμα ανοιχτού κώδικα για τη διαχείριση μικροεφαρμογών και υπηρεσιών βασισμένων σε docker containers  
Η πλατφόρμα Labroom διαχειρίζεται την υποδομή, τους διακομιστές και την κλιμάκωση των Docker containers χρησιμοποιώντας  τις δυνατότητες της τεχνολογίας kubernetes  
 
 .  Περισσότερες πληροφορίες στο [Gitlab](https://gitlab.com/labroom/labroom-server)
